# Project Overview / プロジェクト概要\n\n## Purpose / 目的\nCreate a welcoming, trustworthy clinic homepage that makes online booking obvious while keeping phone access easy.
オンライン予約を分かりやすく、電話もしやすい信頼感のあるクリニックサイトを目指します。\n\n## Audience / 想定ユーザー\n- Existing patients checking hours/access / 既存患者の確認用途\n- New patients seeking services/staff info / 新規患者の情報収集\n- Mobile users needing quick booking/directions / スマホで素早く予約やルートを探す人\n\n## Page Sections / ページ構成\n- Hero with promise, dual CTAs (WEB予約 primary, アクセス secondary), alert for notices\n- 診察時間カード（ステータス＋予約ボタン）\n- おしらせアコーディオン\n- 院長挨拶・診察内容・当院について・理念・スタッフ紹介\n- アクセス/お問い合わせ（地図・連絡先）\n- シンプルなフッター\n- WEB予約ページ (booking.html): 予約フォーム\n- ニュースページ (news.html): 過去のニュースリスト\n- 診察内容ページ (services.html): サービス詳細リスト\n- 当院についてページ (about.html): 医院詳細と画像\n- 理念・方針ページ (policy.html): 医院の理念・方針詳細\n\n## UX Goals / UX目標\n- Clear primary action, consistent buttons / 主要アクションの明確化と統一ボタン\n- Comfortable Japanese typography, balanced spacing / 日本語に適したタイポと余白\n- Responsive nav with mobile hamburger overlay / モバイルでのハンバーガーナビ\n- Accessible focus states, aria-expanded accordions, dismissible alert / フォーカス可視化・ARIA対応・閉じられるお知らせ\n## Navigation behavior
- Global navigation starts between header and hero, then sticks to the top with a translucent backdrop when scrolling down
