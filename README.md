No build or server required. / ブラウザで `index.html` を開くだけ。\n- Assets live under `image/` and are referenced directly. / 画像は `image/` 配下を直接参照。\n\n## Structure / 構成\n- `index.html` – markup and light interactivity / ページ本体\n- `css/style.css` – tokens, layout, components, responsive rules / スタイル\n- `image/` – logo, hero, service icons, staff photos, etc. / 画像一式\n- `docs/` – overview, tech spec, AI prompt, accessibility, deployment, image notes / ドキュメント\n\n## Development notes / 開発メモ\n- Palette: green-light base with blue accents; font is Noto Sans JP via Google Fonts. / グリーン×ブルー基調、Noto Sans JP。\n- Breakpoints: 960px & 600px control stacking and mobile nav. / 960px・600pxでレイアウト/ナビ切替。\n- Accessibility: focus-visible outlines, `aria-expanded` accordions, dismissible alert, nav overlay lock. / フォーカス表示・ARIA対応アコーディオン・アラート閉鎖・ナビオーバーレイ。\n\n## Ownership / 運用\nContent remains in Japanese; no backend.
